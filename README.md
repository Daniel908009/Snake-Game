# Snake-Game
a classic Snake Game made in Python with Pygame library <br>
Game is fully resizable and has a settings window <br>
<h1>Controls</h1>
W, UP ARROW = up <br>
A, LEFT ARROW = left <br>
S, DOWN ARROW = down <br>
D, RIGHT ARROW = right <br>
R, RETURN = reset game <br>
